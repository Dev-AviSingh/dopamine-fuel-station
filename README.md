# Dopamine Fuel Station
A simple React.js + tailwind css app that fetches animal pictures from public apis. 
## APIs used
- [PlaceBear](https://placebear.com/)
- [PlaceDog](https://place.dog/)
- [PlaceKitten](https://placekitten.com/)
